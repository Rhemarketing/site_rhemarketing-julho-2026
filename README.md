# RHE Marketing — Site Julho 2026

Novo site institucional da RHE Marketing, desenvolvido com Next.js, React, TypeScript e CSS responsivo.

## Destaques

- Direção visual “Impacto Digital”
- Apresentação de serviços e portfólio
- Integração com WhatsApp e Instagram
- Layout responsivo para desktop e dispositivos móveis
- Otimização básica de SEO e acessibilidade

## Desenvolvimento local

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```
